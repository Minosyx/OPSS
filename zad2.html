Wzór Eulera

Niech φ ∈ R, zaś i jest jednostką urojoną, wtedy wzór Eulera ma postać:
eiφ = cosφ + isinφ

Tożsamość Eulera,
czyli najpiękniejszy wzór matematyki...

Dla φ = π wzór Eulera przyjmuje postać:
eπi + 1 = 0

Kawałek kodu w języku C


#include <stdio.h>

unsigned silnia(unsigned n)
{
	unsigned i, wynik = 1;

	for (i = 2; i <= n; i++)
		wynik *= i;

	return wynik;
}

int main(void)
{
	unsigned liczba;

	printf("Podaj liczbe naturalna: ");
	scanf("%u", &liczba);
	printf("%u! = %u\n", liczba, silnia(liczba));

	return 0;
}
